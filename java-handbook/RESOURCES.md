# 🛠️ Recommended Java Learning Resources

A list of recommended books, documentation repositories, and profiling tools to advance your JVM capabilities.

---

## 📚 Recommended Books

1. **Effective Java (3rd Edition)** by Joshua Bloch
   * *Best for*: Design patterns, API design, clean code practices, and modern collections.
2. **Java Concurrency in Practice** by Brian Goetz
   * *Best for*: Multithreading, JMM rules, and custom synchronization structures.
3. **Optimizing Java** by Benjamin J. Evans
   * *Best for*: JVM performance, garbage collectors tuning, and profiling strategies.

---

## 📖 Official Documentation & Specifications

* **[Oracle JDK 21 Documentation](https://docs.oracle.com/en/java/javase/21/)**: The official Java 21 documentation.
* **[Java Language Specification (JLS)](https://docs.oracle.com/javase/specs/jls/se21/html/index.html)**: The formal rules of Java syntax, types, and variables.
* **[JVM Specification (JVMS)](https://docs.oracle.com/javase/specs/jvms/se21/html/index.html)**: Structural details of class structures and bytecode instructions.

---

## 🛠️ JVM Profiling & Analysis Tools

* **VisualVM**: Combined visual interface for monitoring heap memory, thread states, and garbage collector latency.
* **JProfiler**: Advanced commercial profiler to find thread deadlocks, database query performance issues, and memory leaks.
* **jcstress (Java Concurrency Stress)**: OpenJDK harness to write tests for verifying concurrent execution outcomes and JMM happens-before guarantees.
