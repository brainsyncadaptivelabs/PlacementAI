# 📅 Structured Study Schedules

Pick the schedule that matches your current preparation timeline.

---

## 🚀 1. The 3-Month Plan (Fast Track / Revision Mode)
Ideal for engineers who already have a CS degree/industry experience and are looking to refresh concepts quickly before interviews.

```
Week 1-3: Advanced DSA -> Week 4-6: System Design (HLD/LLD) -> Week 7-10: OS/Networks/SQL -> Week 11-12: Mock Interviews
```

| Weeks | Focus Area | High-Priority Topics | Check-Ins |
| :--- | :--- | :--- | :--- |
| **Week 1-3** | **Advanced DSA** | Graph traversals, Binary Search, Sliding Window, Heap, Hash map strategies. | Solve 50 Medium/Hard Leetcode problems. |
| **Week 4-6** | **System Design (HLD/LLD)** | Scalability patterns, Caching, DB Sharding, REST vs gRPC, Design Patterns. | Design TinyURL, Uber, and WhatsApp. |
| **Week 7-10** | **OS, DBMS, Networks** | Virtual Memory, CPU scheduling, ACID transactions, TCP handshake, TLS. | Review cheat sheets. |
| **Week 11-12** | **Revision & Mocks** | Behavioral prep, company tags, and timed mocks. | Conduct peer mock interviews. |

---

## 📈 2. The 6-Month Plan (Interview Readiness)
The optimal time frame for candidates targeting Big Tech roles who want a robust foundation.

* **Month 1-2**: Complete Programming Language mastery & fundamental Data Structures.
* **Month 3-4**: Advanced DSA (Trees, DP, Graphs) & Low-Level Design (SOLID, design patterns).
* **Month 5**: High-Level System Design (Microservices, CDN, message queues) & Databases.
* **Month 6**: OS internals, TCP/IP networking, and mock interviews.

## 🎓 4. The 2-Year Plan (Zero-to-Hero Academic Path)
Ideal for self-taught engineers, students, or career switchers starting from absolute scratch.

### Year 1: Foundations & Algorithm Engineering
* **Quarter 1**: Syntax and execution of compile-time languages (C/C++) and JVM languages (Java). Memory layouts.
* **Quarter 2**: Linear Data Structures (Arrays, Linked Lists, Stacks, Queues) and basic algorithms (Sorting/Searching).
* **Quarter 3**: Non-linear structures (BST, AVL, Heaps, AVL/Red-Black Tree theory) & Recursion.
* **Quarter 4**: Dynamic Programming, Advanced Graph theory, and math algorithm proofs.

### Year 2: Systems, Scale, and Industry Mastery
* **Quarter 5**: Operating System kernel mechanics, process control, Virtual memory mapping, Linux command-line, shell scripting.
* **Quarter 6**: Computer Networking (Sockets, epoll multiplexing, OSI layer protocols) & SQL database design.
* **Quarter 7**: Software Engineering practices (Git, CI/CD, Unit testing frameworks) & LLD design patterns.
* **Quarter 8**: Distributed systems HLD (CAP theorem, Paxos consensus, load balancing, message queues).
